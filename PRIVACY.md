# Privacy Policy — realestate-mcp

**Last updated:** March 25, 2026

## Summary

**realestate-mcp does not collect, store, or process any personal data.**

## Details

### Data Collection

- **No personal data collected** — We do not collect names, email addresses, IP addresses, or any other personally identifiable information.
- **No cookies** — The Service does not use cookies of any kind.
- **No tracking** — No analytics, telemetry, fingerprinting, or behavioral tracking.
- **No logging** — API requests are stateless and not logged. No request content, metadata, or usage patterns are recorded.
- **No property data stored** — Property addresses, descriptions, and other details provided in tool calls are processed in-memory and discarded immediately after the response is returned.

### Data Processing

- All MCP tool calls are processed in-memory and discarded immediately after the response is returned.
- No data is persisted to disk, database, or cloud storage.
- No data is transmitted to third parties.

### Third-Party Services

This Service does not make any external network calls. All tools operate fully offline.

### GDPR Compliance

This Service is compliant with the EU General Data Protection Regulation (GDPR) by design: since no personal data is collected or processed, there is no personal data to protect, transfer, or delete.

### CCPA Compliance

This Service is compliant with the California Consumer Privacy Act (CCPA) by design: no personal information is collected, sold, or shared.

### Changes to This Policy

We may update this Privacy Policy from time to time. Changes will be reflected in the "Last updated" date above.
